# 🌍 AI Travel Planner

An intelligent travel planning application that generates personalized itineraries using artificial intelligence.

## ✨ Overview

This project is a full-stack travel planner that leverages AI to create customized travel experiences based on user preferences. It simplifies trip planning by automatically generating optimized itineraries.

## 🚀 Features

- 🤖 AI-powered itinerary generation  
- 📍 Personalized travel recommendations  
- 🗺️ Dynamic trip planning  
- ⚡ Fast and responsive application  
- 🔐 Secure API integration using environment variables  

## 🛠️ Tech Stack

- Backend: Java + Spring Boot  
- AI Integration: Groq API  
- Build Tool: Maven  
- Environment Management: Environment Variables
